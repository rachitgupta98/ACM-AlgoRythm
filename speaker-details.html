Event Details

Hackathon

A problem set will be provided by the company to be solved at the Hackathon.

  • Teams will Develop a Product within the given time frame.
  • Judging at continuous intervals and on various parameters on the day of event.
  • Providing Winners of the Hackathon with Internship offers or Cash Rewards and providing Participation Certificates to the rest of the teams.



Blind Code

The screen will be turned off and then the participant has to solve the given problem statement will the screen is off, the solution with minimum no. of errors wins.



Code Buzz

There will be a pseudo code on the screen, the player who knows the output will press the buzzer and answer. The team answers most wins.



Catch The Flag

Player will be presented with webpage(s), he has to find clues which maybe anywhere in the browser restricted to website content only. Moreover on moving to next pages after finding clues, there could be a pattern which will help in the last page. Scoring will decided by number of pages cleared then by time taken.



Code Hunt

The team will be given a pseudo code which on solving will give the hint to the next location where another pseudo code is hidden, the team which reach the finsh line first will win



Fake ICPC

It will be similar to ICPC i.e. a set of problems will be given to the participants and they wlll have to solve them.



Fit Byte

There will be a team of 2 people one will perform coding and other person will perform any physical activity. If the person performing physical activity stops then code will be erased.



Mock Gate

It will be a Mock Gate Exam.



New Language Programming

A new syntax will be given to the Participant and he has to code according to that.



Game Development Workshop

A theme will be given and according to that the teams have to design a game.
